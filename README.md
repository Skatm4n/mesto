# Mesto

[![Mesto](https://cdn.iconscout.com/icon/premium/png-128-thumb/link-1741098-1482052.png)](https://skatm4n.github.io/mesto/)

## Использовались
- HTML5
- CSS3
- JavaScript

## Что есть?

- Добавлять фотокарточки
- Возмоность посмотреть фото в крупном плане
- Активные лайки
